Hexagon Dots

Game Description

Check out a more completed version in the Links section! Includes a colorblind mode.

Based on the popular mobile game Dots, but with more degrees of freedom! Match each dot as long as they are next to each other!

Controls: Mouse click along the matching dots. (Not compatible with mobile)

About

A short game project made in both Unity and Phaser 3, both within the same week. Working on this project was one of my first experiences with the game engine Phaser 3. Both versions were created with an object-oriented structure.
