Image Gallery

Game Description

Press SPACE to jump! As the Ninja Block, dodge the incoming ninja stars to increase your score! Watch out though, as with each level up, the ninja stars will fly faster and faster!

Controls: Space to jump.

About

A short minigame made using Unity to practice with the engine. The base game was made under an hour. Graphics and polish were added within an hour the following night. The main goal of this project was to see how fast I could make a short game.
