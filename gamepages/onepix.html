Game Description

The evil Pengu army has invaded the world and turned you into a pixel! Now you must venture forth, reclaim your lost pixels, and kick some Pengu butt!

About

A game made under 72 hours for Ludum Dare 45, in collaboration with Branchi.

The theme was "Start with Nothing", so we designed a game where the character would start off as "nothing", and eventually build itself up as the player progresses. This was our first game jam, so a lot of things didn't turn out the way we exactly wanted, but it was still a fun learning experience.
