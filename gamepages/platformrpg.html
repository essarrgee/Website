Game Description

[Project still in progress.] A platform fighting game similar to the iconic Super Smash Bros. franchise. Select from a variety of characters from popular indie titles.

About

A fighting game made to bring together multiple popular Roblox and idie game franchises together. The game started off as an original platformer with combat mechanics, but quickly grew into a multiplayer-focused fighting game. Scripted with Lua in an object-oriented style using Roblox Studio.
