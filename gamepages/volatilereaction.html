Game Description

Place bombs to clear a path to the exit in this puzzle game! 14 levels to clear!

About

Submission for the Roblox Developer Challenge 2023 game jam. Compared to previous Roblox game jams, this game was the most complete in terms of implementation of mechanics. After finishing the first few levels, we had time to add in a few extra levels that further utilized these mechanics.
