BakuDB

App Description

View a list of Bakugan! Sorting, filtering, and more information available soon.

About

A webapp made using HTML, CSS, JavaScript, and Firebase(Hosting/FireStore). The code simply connects to the FireStore database, reads its data, and translates it onto the page using simple front-end.

I chose Bakugan as the subject to practice on since, like Pokemon, there are a large amount of them, each with different stats.
