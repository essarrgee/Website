App Description

Make orders for an ice cream cone at an ice cream store! Or, view the orders as a store manager! (All products, stores, and customers are not real.)

(The database (.db) file must be in the same folder as the application to run correctly.)

About

A java application made with Java for a class project in a team of two. The database was created using SQLite Studio and connected to the java application using JDBC.
